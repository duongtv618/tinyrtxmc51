# tinyrtxmc51

A tiny rtos feature for at89c51

# Specs

MCU     : AT89C51<br>
iRAM    : 128 bytes<br>
Flash   : 4KB<br>

# Compiler

[SDCC](https://sdcc.sourceforge.net/)

# Build

I use **make** on **debian** to build this project<br>
In terminal run:<br> 

    make

This will generate a file main.hex in debug foler
